# viva-mais-app
Aplicativo mobile para criação de hábitos saudáveis com sistema de pontos, persistência de dados e feedback inteligente.

# 📱 Viva+

O Viva+ é um aplicativo mobile desenvolvido com o objetivo de ajudar usuários a criarem e manterem hábitos saudáveis no dia a dia.

---

## 🚀 Funcionalidades

- 💧 Controle de consumo de água
- 🏃 Registro de atividades físicas
- 😴 Monitoramento de sono
- 🏆 Sistema de pontuação (gamificação)
- 💾 Salvamento de dados local
- 🤖 Feedback inteligente baseado no comportamento do usuário

---

## 🧠 Diferencial

O aplicativo utiliza uma lógica de recomendação que analisa as ações do usuário e gera sugestões personalizadas, simulando o uso de inteligência artificial.

---

## 🛠️ Tecnologias utilizadas

- Godot Engine
- GDScript
